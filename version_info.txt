# Correctif de l'audit expert (dimension 20) : sans ce fichier, l'onglet
# "Details" des proprietes Windows de l'executable distribue n'affichait
# AUCUNE metadonnee (FileVersion/ProductVersion a 0.0.0.0, CompanyName/
# FileDescription/LegalCopyright vides) - moins de professionnalisme percu et
# un signal generique de plus (parmi d'autres, deja documentes dans le
# README) pouvant renforcer a tort la mefiance SmartScreen/antivirus sur un
# executable PyInstaller non signe.
#
# IMPORTANT : filevers/prodvers et les chaines FileVersion/ProductVersion
# ci-dessous doivent etre mis a jour EN MEME TEMPS que APP_VERSION dans
# gui.py a chaque nouvelle version (pas de generation automatique - ce
# fichier est lu par PyInstaller via un mini-parseur dedie, pas par
# l'interpreteur Python, il ne peut donc pas importer gui.py directement).
